--- layout: post title: Useful Resources for Developers title_description: This page will be to keep track of useful resources I've found that I think would benefit anyone who uses them. ---

Books

Software Development

  • The Pragmatic Programmer: From Journeyman to Master - Andrew Hunt & David Thomas
  • The Passionate Programmer - Chad Fowler
  • The Mythical Man-Month: Essays on Software Engineering - Frederick P. Brooks Jr
  • Code Complete, Second Edition - Steve McConnell
  • The Algorithm Design Manual - Steven Skiena
  • Algorithms to Live By: The Computer Science of Human Decisions - Brian Christian & Tom Griffiths

Educational

  • The Ten-Day MBA 4th Ed.: A Step-by-Step Guide to Mastering the Skills Taught In America's Top Business Schools - Steven A. Silbiger

Creative

  • You Can Draw in 30 Days: The Fun, Easy Way to Learn to Draw in One Month or Less - Mark Kistler

Websites

  • Firecode - Practice Java, C, C++, and Python interview problems.
  • Free Code Camp - Learn front end and full stack development.
  • Dev Docs - Combines multiple API documentations in a fast, organized, and searchable interface.
  • Cybrary - Free Cyber Security training including an ethical hacking course.
  • Music For Programming - Large amount of background music to listen to while programming.
  • Learn You a Haskell for Great Good! - Free guide for learning Haskell and functional programming.

Podcasts

  • Programming Throwdown - Programming Throwdown attempts to educate future Computer Scientsts and Software Engineers on a cavalcade of programming and tech topics.
  • Developer Tea - A 10(ish) Minute Podcast for Developers.
